网格的基函数

每个单元的基函数，由在质心的Taylor展开得到

参数
i基函数选择
xi变量参考空间横坐标
eta变量参考空间纵坐标
返回
double 返回结果：
  • i = 0, Psi = 1
  • i = 1, Psi = xi - xi_c
  • i = 2, Psi = eta - eta_c
  • otherwise, no return (only 2nd-order).
